# 2026-07-07

## Shader 重大更新

### Glass.shader 重写
- Shader 名从 `Unlit/EffectShader` 改为 `RoseWindow/Glass`
- 新增 ForwardBase + ForwardAdd + ShadowCaster 三个 Pass
- 添加 Blinn-Phong 光照 + 球谐环境光 + 菲涅尔反射
- 添加四张贴图支持：
  - `_MainTex`：Albedo 颜色贴图（默认 "white"，带 Alpha 通道用于 Cloud 裁剪）
  - `_HeightMap`：高度/位移贴图（默认 "black"，Domain Shader 中沿法线位移）
  - `_BumpMap`：法线贴图（默认 "bump"，TBN 矩阵转到世界空间）
  - `_RoughnessMap`：粗糙度贴图（默认 "white"，控制 Blinn-Phong 高光锐度）
- 预留 `_PlayerPos` 和 `_PlayerInfluence` 隐藏参数供未来扩展
- 贴图缺失时不会报错（Unity 内置默认纹理 fallback）

### Cloud.shader 新建
- Shader 名：`RoseWindow/Cloud`
- 管线：VS → HS → Tessellator → DS → GS → PS
- 核心特性：
  - 粒子永远不凝结（`_BaseSpread > 0`）
  - Alpha 裁剪保证窗形轮廓（透明区域不生成粒子）
  - 三角形内随机采样避免网格感（`SV_PrimitiveID` + hash）
  - GS 内从三角形边反推 TBN，采样法线贴图
  - PS 做完整 Blinn-Phong + 球谐环境光 + 菲涅尔
  - 玩家排斥散射（二次衰减）+ 弹簧回弹 + 布朗漂移
  - ForwardBase + ForwardAdd 双 Pass
  - Billboard 面片（始终面向摄像机），圆形软边遮罩

### RoseWindowCloud.cs 新建
- 路径：`Assets/Scripts/RoseWindowCloud.cs`
- 每帧向 Cloud.shader 推送玩家位置和散射参数
- 预缓存 Shader.PropertyToID（避免字符串查找开销）
- OnValidate 实时响应 Inspector 修改
- 自动创建材质实例并清理

### 多物体组合问题（框架+玻璃分离模型）
- 用户的花窗由 Frame + Glass 两个独立 GameObject 组成
- Glass.shader 可直接分别使用，无问题
- Cloud.shader 的 Alpha 裁剪在分体 mesh 上会导致粒子轮廓断裂
- 推荐方案：额外创建一个 CloudProxy Quad（完整贴图）专用于粒子云模式
- Frame/Glass 实体在云模式下隐藏，CloudProxy 在实体模式下隐藏
- 已补充到操作文档 3.5 节
